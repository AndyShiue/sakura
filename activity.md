# 物語

物語是不限時間隨時開放的關卡  
目前的物語分成三部分：  
**主線**、**靈揀會**和**後日談**  

建議新手玩家在開啟靈揀會後可以轉戰靈揀會  
因為靈揀會可以刷到好用的怪物卡  
要是不刷怪物卡  
主線很快就會卡關了  

## 主線

主線的劇情很豐富很精彩  
目前一共有十六章  
一路上送的好東西也不少  
但是沒辦法刷到SSR或SSSR卡  

主線到了第七關難度會變難不少  
接著難度慢慢上升  
到了16-4時難度再一次大幅上升  

過主線5-5能解鎖角色西園寺綾乃  
過6-7能解鎖雨宮夏希  
過14-8能解鎖早乙女葵  

## 靈揀會

靈揀會基本上沒有劇情  
在主線通過4-6時開啟  
目前一共有7個Mission  
每個Mission又各有十小章  
每張有八小關  

其中  
M3開始每兩關送20水晶  
M4-10-8,M5-9-8,M5-10-8,M6-9-8,M6-10-8,M7-9-8,M7-10-8這幾關可以刷章  
M7每四關會送一張煉化素材  
刷章的意思就是重複刷掉落的SSR卡拿去賣換紋章  
另外靈揀會的不少SSR或SSSR練上去也是有用的  
(順帶一提 底下這張卡一般大家叫作*雷胖*  
能在M5-9-8刷到 算是比較快能拿到而且又好用的SSSR王卡)  

![雷胖](http://sk.wiki.1758play.com/images/e/ed/%E6%83%A1%E9%AD%94%E4%BC%AF%E7%88%B5%EF%BC%8E%E5%B8%9B%E7%8E%9F%EF%BC%88%E4%BA%94%EF%BC%89.jpg)

靈揀會某些關卡蒐集完成獎勵是特殊卡*吉籤*  
裝備上去走路能增加經驗值  
要是沒有更好的特殊卡請務必拿到  
*末吉籤*能在M1-4-1拿到 走路經驗+4  
*半吉籤*能在M3-7-8拿到 走路經驗+8  
*小吉籤*能在M6-7-8拿到 走路經驗+12  

![小吉籤](http://sk.wiki.1758play.com/images/7/79/%E5%B0%8F%E5%90%89%E7%B1%A4.jpg)

已經有*競選回憶錄*的人就不用管末吉半吉了  
這之前活動出的  

![競選回憶錄](http://sk.wiki.1758play.com/images/3/3e/%E7%AB%B6%E9%81%B8%E5%9B%9E%E6%86%B6%E9%8C%84II.jpg)

## 後日談

和女角告白後會開啟那個女角的後日談  
目前有六個女角  
所以後日談也有六個  
每個角色的後日談各有兩章  

後日談有少量劇情  
每關需要的體力多  
無法刷SSR或SSSR怪物卡  
但中途能蒐集**桔梗紋章**  
桔梗紋章可以換八神SSSR卡  
通常都把那張卡叫做*後日八神*  
算是滿不錯的卡片  
只要跑完兩個角色的後日談就能拿到超過六張的後日八神了  

![後八](http://sk.wiki.1758play.com/images/0/08/%E5%85%AB%E7%A5%9E%E7%87%90%EF%BC%88%E4%BA%94%EF%BC%89.jpg)

# 活動

嘛 活動 就是限時活動  
所以不是隨時開放的  
活動有分可重複和不可重複的  
可重複的活動跑完後會從第一關重新開始  

有時候會有不定期的特別活動  
例如通常出限時轉蛋都會開刷限時轉蛋的活動  
詳情請注意落櫻粉絲團  
另外還有些活動是完全不會公告的  

## 夢境

每周周一到周五都會有一個夢境活動  
0點開始 23點結束  
其中  
周一是椎名奇幻的夢境  
周二是西園寺優雅的夢境  
周三是雨宮神秘的夢境  
周四是千櫻院華麗的夢境  
周五是佳奈謎樣的夢境  
奇怪為什麼只有佳奈是名字xD  

夢境是不可重複活動  
完成活動能拿到一張SSR卡  
一般叫它**夢境卡**  

因為夢境卡是SSR卡  
實力不如靈揀會的SSSR卡  
還沒辦法順便刷章  
而且要穩五階需要的時間長  
所以不太建議刷夢境  

![夢椎](http://sk.wiki.1758play.com/images/4/48/%E6%AD%A6%E9%AC%A5%E5%A5%B3%E5%AD%A9%EF%BC%88%E4%BA%94%EF%BC%89.jpg)

## 周末活動  

以下活動的開始時間都是周六0點  
結束時間是周日23點  
另外  
屬性試煉、古卷、賞金三個活動是每周末輪流開啟的  
要是能過活動最高難度  
建議每周末都用來打活動  

### 禮物

禮物是每周末都會有的活動  
分成初階、上階兩個難度 可重複  
跑上階比跑初階划算  
跑禮物能獲得**三葉紋章**  
而三葉紋章能用來兌換給女角的禮物  

### 屬性試煉

每三周會有一次屬性試煉  
屬性有可能是地、水、火、風、雷其中之一  
舉例來說假設是火之試煉  
那那次活動的怪物就都會是火屬  

跑屬性試煉能獲得粉卷和藍卷  
一共分成初階、上階、超上階三個難度  
其中超上要是完成還能得到一張SSR萬用卡  
不可重複  

![藍萬用](http://sk.wiki.1758play.com/images/1/1a/%E5%8D%83%E5%B9%B4%E8%A9%9B%E5%92%92IV%EF%BC%88%E4%B8%80%EF%BC%89.jpg)

建議每次的屬性試煉都要跑  
除非是和其它活動衝到  

### 古卷

每三周會有一次古卷  
跑古卷能獲得大量經驗石  
一共分成初階、上階、超上階三個難度  
不可重複  

### 賞金

每三周會有一次賞金  
跑古卷能獲得大量金錢  
一共分成初階、上階兩個難度  
不可重複  

## 其它活動

以下活動都是不定期舉行  
而且都有和一般物語、限時活動不一樣的玩法  

### 活動王

```
1.活動一共八關，可重覆探索，首次探索完成時，可獲得關卡獎勵。
2.探索過程中，會隨機遇上魔王，依戰鬥對魔王造成的傷害給予對應積分，活動結束後依積分排名給予獎勵。
3.每次戰勝魔王後，魔王等級會提升，魔王亦可以請好友協助幫打。
4.與自己遭遇的魔王進行戰鬥，每次消耗行動能量10。與好友遭遇的魔王進行戰鬥，每次消耗行動能量5。
5.打死魔王時，會獲得額外的積分加成。
6.自己遭遇的魔王被好友打死時，自己也會獲得額外積分。
7.只要參加活動即可獲得一份參加獎。
```

活動王除了參加獎還有另外兩種獎勵  
分別是積分獎和排名獎  
建議每次活動王都打到一千萬分  
對大多數玩家來說一千萬分應該都不是太難的事情  

活動王三億分能拿到SSSR萬用  
想挑戰的玩家能試試看  

### 應援團

```
1.活動開始時系統會隨機分配所屬應援團。
2.活動前2天為盲選，於此時可更換1次應援團，盲選第2天開始進行對戰，此時不會看到應援團分數。
3.每次對決會隨機與應援團成員組隊並可變更活動牌組。
4.連勝越多，獲得『可貢獻分數』越高，最高至20連勝。
5.可貢獻分數需進行應援貢獻，將分數加至所屬應援團所為實資分數。
6.戰敗時。身上可累積貢獻分數立即歸零。
7.活動結束，第一名應援團全體獲得應援團大獎；各應援團依內部排名獲得獎勵（限制:積分達1分以上) 重新調整積分加成，於活動中不定時出現，請密切注意。 
```

應援團每次對決需要耗費金錢  
是個燒錢的活動  

### 機構戰

機構戰不是個人和個人間的對決  
而是機構和機構間的對決  
機構戰需要耗費大量金錢  
分成兩階段  
第一階段前十六名會晉升第二階段  

#### 機構席次戰 

```
報名規則:
1.由機構的總長、副長消耗機耗資源報名活動。
2.報名時，機構至少1位副長存在
3.報名順序先搶先贏，若最後報名的機構為奇數順位，則取消資格並退回以消耗資源。

賽程規則：
1.報名時間為第1天12:00至第3天12:00。
2.第1場對決時間為第3天18:00至20:00。
3.第2場對決時間為弟3天21:00至23:00。
4.第3場對決時間為第4天18:00至20:00。
5.弟4場對決時間為第4天21:00至23:00。

活動規則：
1.活動分成前之陣、裏之陣、後之陣三陣。
2.成員須先決定於哪一陣參戰，才得以出戰。
3.參戰後，便無法換陣，於對決時間內才得以出戰。
4.成員於選定的陣行進行戰鬥來獲取積分。
5.於對決結束時，若三陣終有2陣獲勝，可獲得額外機構總積分。
6.席次戰共4場對決，成員須於4場內盡可能地衝高機構總積分。
7.席次戰結束後，取積分最高的16強機構進入下一階段活動。

活動獎勵：
1.活動結束，按機構最終積分、排名給與全體成員獎勵。
2.獎勵獲得需滿足參加獎條件:40積分。
```

#### 機構王者戰

```
資格規則：
1.由席次戰決定出的前16席次進行王者機構的爭奪。

賽程規則：
1.開始時間為00:00。
2.八強決定賽(第1輪)為11:00至13:00。
3.四強決定賽(第2輪)為14:00至16:00。
4.準決賽(第3輪)為17:00至19:00。
5.冠軍戰、季軍戰為20:00至22:00(同步進行)。

活動規則：
1.同席次戰，分成前之陣、裏之陣、後之陣三陣。
2.成員需先決定於哪一陣參戰，才得以出戰。
3.參戰後，便無法換陣；於對決時間內才得以出戰。
4.成員於選定的陣進行戰鬥來獲取積分。

獲勝條件：
1.機構需於三陣當中取得兩陣才得以勝出。

活動獎勵：
1.活動結束，按機構最終排名給予全成員獎勵。
2.成員按個人累計總積分獲得對應獎勵。
3.獎勵獲得需滿足參加獎條件：40積分。
```
